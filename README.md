## âš™ï¸ How to Run Locally

If you'd like to test this project on your own machine:

1.  **Clone the repo:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/expense-tracker.git](https://github.com/YOUR_USERNAME/expense-tracker.git)

    ```

2.  **Setup Backend:**
    ```bash

    npm install
    # Create a .env file with PORT, MONGO_URI, and JWT_SECRET
    npm start
    ```

3.  **Setup Frontend:**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
